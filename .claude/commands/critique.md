# /critique

Identify blind spots in the most recent decision, design, or direction using the following angles.

## Do

1. **Overlooked risks** — What could cause this approach to fail?
2. **Hidden assumptions** — What is being assumed as true but not yet verified?
3. **Strong alternatives** — What other approaches were not considered?
4. **Worst-case scenario** — What does the most painful failure look like?

## Don't

- Offer praise
- Close with "you're absolutely right"
- Repeat what the user already said

## Output format

Bullet points for each angle, concise. Attach severity (high / medium / low).
End with a single sentence: "The single most important point is…"
